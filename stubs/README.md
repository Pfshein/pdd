# stubs/ — детерминированные заглушки для тестов

Фейковые внешние зависимости, чтобы ядро тестировалось без реальной модели и процессов.

## Contents

| Файл | Что заменяет |
|---|---|
| `qwen_stub.py` | Долгоживущий agent-процесс. Спавнит ребёнка-сонника, пишет его pid в `argv[1]`, сам спит. |

## Purpose

`qwen_stub.py` существует ради одного инварианта: при таймауте стадии `run_process` обязан
убить **всё дерево процессов**, а не только верхний. Стаб даёт предсказуемое
родитель→ребёнок дерево, по которому тест проверяет kill-tree (см.
`tests/test_killtree.py`) без запуска настоящего qwen.

## Invariants & gotchas

- Стаб должен оставаться **тупым и детерминированным** — никакой сети, никакой логики модели.
  Если тесту нужно «поведение модели», мокай ответ в тесте, а не усложняй стаб.
